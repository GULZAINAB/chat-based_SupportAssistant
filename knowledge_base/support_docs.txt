Wi-Fi Troubleshooting:
1. Ensure Wi-Fi is turned on.
2. Restart your router.
3. Reconnect to the network.
4. Run the network troubleshooter.

System Restart:
1. Click Start menu.
2. Select Power > Restart.
3. Wait for the system to reboot.

Change Date and Time:
1. Go to Settings > Time & Language.
2. Turn off "Set time automatically".
3. Set time manually.

Clear Cache:
1. Open your browser settings.
2. Go to Privacy > Clear Browsing Data.
3. Select "Cached images and files" and clear it.
